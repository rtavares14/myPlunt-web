# Plunt — Frontend

React 19 + TypeScript + Vite + MUI v9 + Tailwind CSS v3.

See the [root README](../README.md) for full project setup and context.

## Commands

```bash
npm run dev      # Dev server → http://localhost:5173
npm run build    # Type-check + production build
npm run lint     # ESLint
```

## Colour palette

Edit `tailwind.config.js` — changes propagate to all Tailwind classes and (via the comment pointer) to the MUI theme in `src/App.tsx`.
